# 你好! 我是京衡。

这里将呈现我在学习前端的过程中遇到的问题以及解决。

# 代码编辑器

这里的编辑器，也是IDE，是一个集成开发环境，可以说是“高级代码编辑器”。

我使用的微软的VSCode，相对轻量，社区拥有丰富插件按需添加。

另一个是HBuilder，用来写uniapp的。

# 文本编辑器

这里的编辑器，使用的是makedown文本语法，简约方便，是程序员编辑文本更加广泛的选择。

我使用的软件是Typora，是目前编辑makedown文本的主流软件。

（听说正式版本开始收费了，我的typora使用的是beta版0.9.86目前没有提示更新和收费，且用着，毕竟也用了多年。）

# 代码编译

谷歌浏览器+Edge（开发工具+若干插件，配合使用）

# 技术栈

关于前端，我目前学习的技术栈包括HTML、CSS、JavaScript、Node、Vue以及一些衍生出来的技术面。

目前会一直学习大前端的技术，后期会在游戏或者WebGL（three.js）中选择一个方向继续发展。

# 搜索力与社区

我对于前端填坑的解决策略：检测与搜索

检验问题出在哪个环节如何定义这个“坑”，

根据定义的“坑”在社区里搜索解决方案，这里的社区即程序员交流群体，网站平台（掘金、哔哩哔哩、GitHub、博客园、CSDN……），以及百度、微信/QQ好友群体
